世界観

大切な友と日常を守るため、キミは現実する噂へと挑む！！


現実を侵食する「魔」にい染まりし異端を狩る先兵たる「猟犬(ハウンド)」。
一般社会とかけ離れた世界に生きる猟犬のキミが、唯一「普通の人」として生きることを許された大切な場所「星桜学園」では、約半年前から奇妙な現象が起きていた。

それは「噂の現実化」

荒唐無稽な噂であっても、それが広まれば「本当におこってしまう」怪奇現象だ。
だが、この非常識な状態を、学園の生徒は「普通」だと思い込んでいた・・・。

すべての原因が学園に潜む「魔」が展開する結界「歪」の影響であることを知ったキミは、在学中は封印し続けるはずだった猟犬としての「力」の一部を開放し、学園の日常を壊さぬよう気をつけながら歪の解消を目指す。

システム的な特徴

普通のTRPGで必要なのはダイスですが、このゲームではトランプを使います。
実際プレイした感想では、学園異能モノの殻をかぶった、カードを使うボードゲームみたいなものって感じになります。

このゲームにおけるリソースは、自分の「噂ゲージ」、「ストレスゲージ」、「フレンド」です。この3つをうまくやりくりして、シナリオクリアするゲームともいえます

噂ゲージ PC間で共有 シナリオクリア判定に影響する最重要ステータス。シナリオ終了時に0になっていないとゲームオーバー
ストレスゲージ 各PC毎 PCの持つHPみたいな感じ。シナリオ終了時に限界突破したままでいるとPCロスト
フレンド 各PC毎 他のゲームでいうところのアイテムに近いかも。もしものときのために大事にしよう

オートセッションについて

このゲームには全てランダムイベントで遊ぶことができる「オートセッション」がついています。

  • シナリオを作らなくても遊べる
  • 何度も遊んで、PCの封印解除レベルを上昇させることができる
  • ランダムに登場するNPCとフレンドになることができる
  • PC同士が協力すれば、GMがいなくてもプレイが可能

オートセッションでは、PCは歪によって噂が顕在化して起こる特異な事件が、さらなる噂を呼ぶ前に、急いでこれを解決しなければならない。そうしなければ、噂が事件を起こし、事件が噂を作り、という連鎖が膨れ上がり、やがて歪みが生み出す日現実によって現実の世界があふれかえってしまうだろう。そうなればPCも学園に通う友人も助からないだろう

勝利条件

「歪を顕在化させることなく1週間をやり過ごすこと」

敗北条件

「退学」... 自PCのストレスゲージが振り切れて廃人になる

「歪の顕在化」 ... 1週間経過後に噂ゲージが0になっていない

判定

同値以上で成功

判定後、手札を消費(してトラッシュに送る)したPCは、山札から1枚ドローして手札に加える

難易度 達成値
基本 イベントの判定値 PCの能力値 + 手札の数字
「情報収集」使用時 イベントの判定値 -2 PCの能力値 + 手札の数字
「支援」使用時 イベントの判定値 PCの能力値 + 手札の数字 + 他のPCの手札の数字
「友情ブースト」使用時 イベントの判定値 PCの能力値 + 手札の数字 + フレンドの友情値
例:
【イベント】 「ああ、窓に窓に！」
【目的】 あれは名状しがｔ・・・いかん、あぶないあぶない・・・
【難易度】 学力15
【クリアポイント】 1

PC1の「学力は5」、PC1の手札の中には「ハートのA」がある。
PC1は、この手札を切ることで、5+15=20でクリアポイント1を達成した。

支援

前提1 支援したPCの行動が終わっていない

利点1 判定に「+支援したPCが提出した手札の数字」

欠点1 支援したPCの行動が終了する

友情ブースト

前提1 イベントにPCのフレンドが関わっている

前提2 PCのストレスが上限値を超えていない

利点1 判定に「+そのフレンドの友情値」

欠点1 山札から手札に1枚カードを引き、PCのストレスとして受け取る

欠点2 判定に失敗した場合、そのフレンドに対する「印象が消失する(フレンドでなくなる)」

情報収集

前提1 イベント開始時であること

利点1 次のターンのイベントで設定された「達成値-2」

欠点1 そのターンは一切の行動ができない

インスピレーションチェック

ネームドNPCをフレンドにすることができる

チェックによりつく「印象」は「NPCからPCへの印象」というニュアンスで、ゲームのシステム上一度つくとなかなか変更できない

手順

  1. GMが山札の上からカードを1枚引く(裏向き)
  2. PCは手札からカードを1枚出す(裏向き)
  3. 全員一斉に表向きにする
  4. 印象を決定する
    • スートが同じ = 印象「GOOD」
    • スートの色が同じ = 印象「NORMAL」
    • スートの色が違う = 印象「BAD」
    • ジョーカー(PC) = カードを引いたそのPCは、好きな印象を選ぶ
    • ジョーカー(GM) = 参加した全PCは、好きな印象を選ぶ
  5. 使用したカードをトラッシュへ送る
  6. PCは1枚ドローする

フレンドチェック

前提1 幕間であること

利点1 成功することにより「フレンドマーカー+1」

欠点1 特になし

「印象」によって難易度が変動する

印象 難易度 達成値
GOOD フレンド基本難易度 -2 PCの能力値 + 手札の数字 + フレンド友情値
NORMAL フレンド基本難易度 ±0 PCの能力値 + 手札の数字 + フレンド友情値
BAD フレンド基本難易度 +2 PCの能力値 + 手札の数字 + フレンド友情値

偶然の出会い

行動消費することで、ランダムなNPCとのイベントを発生させることができる

前提1 幕間であること

利点2 フレンドマーカーを持っていないNPCの場合、「インスピレーションチェック」ができる

利点1 フレンドマーカーを持っているNPCの場合、「フレンドチェック」ができる

欠点1 特になし

噂ゲージ

学園内に噂(によって生じた歪)がどのくらい広まっているのか表すもの

全PC共通、セッション終了時に1ポイントでも残っていると全PCの正体がばれる(ゲームオーバー)

増加してしまう条件

山札から増加枚数分のカードを噂ゲージに送る

スキル使用時 コスト枚数分
イベント失敗時 PC人数x2

減少させる方法

噂ゲージから減少枚数分だけトラッシュに送る。いずれも行動は消費せず、いつでも行える

PCが噂ゲージを引き取る 任意枚数。引き取ったカードに書かれている数字の分、PCのストレスゲージが増加
フレンドマーカーを1つ減らす フレンドマーカーを減らした分だけ

ストレスゲージ

PCが抱えている歪への精神的苦痛

PC個人、セッション終了時に許容量を超えているとそのPCは退学(ゲームオーバー)

増加してしまう条件

PCが噂ゲージを引き取る 任意枚数。引き取ったカードに書かれている数字の分、PCのストレスゲージが増加

減少させる方法

スキル使用 ※効果はスキルによる
フレンドエフェクト使用 ※効果はエフェクトによる

フレンド

フレンドエフェクト、パートナーエフェクト(今回は使用しない)、噂ゲージの減少ができる

フレンドの作り方

方法1 キャラクター作成時の「初期フレンド」

方法2 イベントに登場しているNPCがいる場合の「インスピレーションチェック」

フレンドマーカー

セッション中に、最大で3までためることが可能

増加させる条件

幕間の「フレンドチェック」で成功する

減少してしまう条件

フレンドエフェクト使用
噂ゲージ1つの無効化

友情値

セッション終了後、フレンドマーカーのうち残っている値分上昇する
